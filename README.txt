Photo Album Console App

This app prints photo titles and id's from a photo album api

Python 3.7.3
https://docs.python.org/3.3/using/windows.html

This program and test file can be run in any IDE that uses Python 3.7.3.

If using the command line make sure python 3 is installed properly then enter the following command
  python3 PhotoAlbumTest.py
  python3 PhotoAlbumPresenter.py 
  
 The first line will run the test file and the second will run the application.
